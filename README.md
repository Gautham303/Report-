# CSA Annual Report Templates

This repository contains professionally designed Microsoft Word templates that replicate the design elements, layout, and formatting of the CSA Annual Report PDF.

## 📁 Repository Contents

### Templates
- **`CSA_Annual_Report_Template.docx`** - Basic version with core sections
- **`CSA_Annual_Report_Template_Enhanced.docx`** - Comprehensive version with additional sections

### Documentation
- **`TEMPLATE_DOCUMENTATION.md`** - Usage guide for basic template
- **`ENHANCED_TEMPLATE_DOCUMENTATION.md`** - Comprehensive guide for enhanced template
- **`README.md`** - This overview document

### Source Code

## 🚀 Quick Start Guide

### Option 1: Use Pre-built Templates
1. Download `CSA_Annual_Report_Template_Enhanced.docx`
2. Open in Microsoft Word 2016 or later
3. Follow "DESIGN NOTE" instructions throughout the document
4. Replace placeholder content with actual information
5. Apply manual formatting as specified in documentation

### Option 2: Generate Custom Template
1. Install Python 3.7+ and required packages:
   ```bash
   pip install python-docx pillow
   ```
2. Run the generator script:
   ```bash
   python create_enhanced_template.py
   ```
3. Customize the generated template as needed

## 🎨 Design Specifications

### Color Palette
```
Primary Blue:    #2E3A87 (Dark Blue)
Secondary Blue:  #007BBF (Light Blue)
Accent Teal:     #009688 (Teal)
Text Primary:    #000000 (Black)
Text Secondary:  #808080 (Gray)
Background:      #FFFFFF (White)
```

### Typography
```
Headers:         Arial, Bold, 48pt/36pt/24pt/18pt
Body Text:       Calibri, Regular, 11pt, 1.15 line spacing
Captions:        Calibri, Bold, 9pt
Tables:          Calibri, 10-11pt
```

### Layout Standards
- **Page Size**: A4 (8.3" × 11.7")
- **Margins**: 1 inch on all sides
- **Line Spacing**: 1.15 for body text
- **Image Quality**: 300 DPI minimum
